# Day 2 - Salesforce Platform Basics

1. What is Salesforce Platform?
Salesforce is a cloud-based CRM platform used to manage customer data and applications.

2. Explain:
- App: Collection of tabs and features.
- Object: Stores data in Salesforce.
- Tab: Used to access objects and features.

3. Difference: Configuration vs Coding
- Configuration uses clicks without coding.
- Coding uses Apex for advanced functionality.
